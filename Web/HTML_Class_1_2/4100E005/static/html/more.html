差異

據說在中世紀的歐洲（5～15世紀），騎士們因頂盔披甲馳騁疆場而難以辨別敵友，於是就在鎧甲或盾牌上標上能一目了然的符號，這就是紋章的起源。紋章是授予騎士的，是僅有王室貴族才能擁有的代表「個人」的標誌，具有強烈的象徵權利與統治的含義。另外，紋章還是用以辨別個人身份的象徵符號，因此禁止製作完全相同的圖案。例如在英國，至今仍然由叫做紋章院的政府機構嚴格管理紋章的事務。

由於紋章僅代表個人，所以即使是父母或兄弟姐妹也要使用不同的紋章。與此相反，日本的家徽代表著「家」或「家庭」，因此不僅父母或兄弟姐妹，甚至整個家族都可以使用同一個家徽。關於日本家徽的產生有諸多說法，其中以在以天皇為中心的貴族治國的「平安時代」（794～1185年）末期貴族已開始使用為最有力的說法。
