起源

日本的家紋起源自平安時代，公家貴族作為交通工具的牛車上所繪製的紋樣。起初只是單純的裝飾，後來逐漸演變成識別身份與家系的符號。平安末期至鎌倉時代，武士成為政治舞台的主角後，各武家也開始使用家紋，並把這種文化發揚光大。
